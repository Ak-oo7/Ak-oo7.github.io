# Abhinav I - Personal Portfolio
A modern, premium, responsive personal portfolio website built with **HTML5**, **CSS3**, and **Vanilla JavaScript**. Designed with an Apple/OpenAI/Stripe/Linear-inspired aesthetic using glassmorphism, gradients, smooth interactions, and GitHub Pages compatibility.
## Features
- Dark mode by default
- Light/Dark theme toggle
- Apple/OpenAI inspired UI
- Glassmorphism cards and premium gradients
- Responsive layout for desktop, tablet, and mobile
- Animated hero section
- Typing animation
- Scroll reveal animations
- Smooth scrolling
- Parallax-style floating orbs
- Particle background
- Project filtering
- Animated skill bars
- Custom cursor
- Scroll progress bar
- Back-to-top button
- Loading screen
- GitHub stats cards
- Dynamic GitHub repository fetching
- SEO optimized meta tags
- Open Graph tags
- Accessible structure
- Fast-loading static deployment
